📝

HWP 바로 붙여넣기

생성된 HTML 문서를 한컴 한글에 그대로 붙여넣으면 서식이 유지됩니다

📥

Markdown 다운로드

.md 파일로 내보내어 다양한 편집 환경에서 활용할 수 있습니다

🔒

개인정보 안전

모든 데이터는 브라우저에만 저장됩니다. 외부 서버로 전송되지 않습니다

📎

파일 첨부 분석
